hexo 使用 SSH 部署

2016年3月14日

使用https提交git往往都需要你重复键入用户名和密码，实在太烦，与其这样还不如给配置SSH一劳永逸，下面动手；

配置ssh key

先确认你机器上是否配置了SSH
前往 ~/.ssh 目录查看是否有文件，有的话跳过这节

没有的话先成SSH

指定保存的目录，直接回车，保存当前目录

配置Github

打开 ~/.ssh/id_rsa.pub 复制 至 github New SSH Key

测试

再配置git提交信息

检查一下，是否进行了相应的更改

更改https为SSH

记得修改 hexo _config.yml 文件，仓库地址
